# Haute Sosiety

Haute Sosiety is a premium fashion editorial website built with HTML, CSS, and JavaScript.

## Overview

The project showcases iconic luxury fashion brands with a modern black-and-white aesthetic, smooth motion, responsive layouts, and interactive UI details.

## Features

- Luxury-inspired hero section
- Brand cards for Chanel, Louis Vuitton, Gucci, Prada, and Dior
- 2024-2025 fashion news and trends section
- Popular items showcase
- Brand filtering
- Expandable content blocks
- Smooth scrolling and animated reveals
- Responsive mobile-first improvements

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Structure

- `index.html` - page structure and content
- `styles.css` - layout, theme, responsiveness, and animations
- `script.js` - interactivity and UI behavior

## Run Locally

Open `index.html` in your browser.
